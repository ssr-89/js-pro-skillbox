Что такое классы?

Основные характеристики классов

  1. Определение класса

    Классы определяются с помощью ключевого слова class.

    Например: 

    class Animal {
  constructor(name) {
    this.name = name;
  }
      
  speak() {
    console.log(`${this.name} издает звук.`);
  }
}

    Экземпляры класса

    Для создания экземпляра (объекта) класса используется оператор new: 

    const animal = new Animal('Собака');
animal.speak(); // Вывод: "Собака издает звук."

    Встроенные классы в JavaScript:

    Date, Promise, Error, SyntaxError, TypeError

  2. Конструктор

    Конструктор - это специальный метод, который вызывается при создании экземпляра класса.

    В этом методе можно задать начальные значения для свойств объекта.

    JavaScript поддерживает наследование через классы с помощью ключевого слова extends. При этом дочерний класс может переопределять методы родительского класса и добавлять свои собственные.

  7. instanceof — это оператор в JavaScript, который используется для проверки, является ли объект экземпляром определённого конструктора или класса. Он позволяет определить, принадлежит ли объект определённому типу (или классу) на основании его прототипного наследования.

Члены класса

Свойства — значения, связанные с объектом.

Методы — операции, которые можно производить над объектом.

Геттеры/сеттеры — методы, которые выглядят как свойства. Выполняются при получении (геттеры) и присвоении (сеттеры) значения как в свойство объекта.
